jQuery.Huispinner是一个用js编写的jQuery插件，通常用在电商网站的购物车数量加减位置。可以设置最小值、最大值、递增递减（步长值），可手动输入数字，支持（上、下）方向键实现加、减。

使用方法

1、jQuery.Huispinner.js依赖jquery或者zepto.js。

js调用

$("#Huispinner-demo1").Huispinner(
	{
		value:3,
		minValue:5,
		maxValue:99,
		dis:1,
	},
	function(){
		alert("回调");
	}
);

demo效果

相关参数

属性 默认值 描述 备注
value 1 文本框默认值
minValue 1 最小值
maxValue 999 最大值
dis 1 每一步增量

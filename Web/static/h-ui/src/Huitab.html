jQuery.Huitab是一个用js编写的jQuery插件，实现选项卡效果。

使用方法

1、jQuery.Huitab.js依赖jquery或者zepto.js。

js调用

$("#HuiTab-demo1").Huitab({
	index:0
});
$("#HuiTab-demo2").Huitab({
	index:2
});
$("#HuiTab-demo3").Huitab({
	tabEvent:"mousemove",
	index:0
});

相关参数

属性 默认值 描述 备注
tabBar '.tabBar span' 触发区域
tabCon ".tabCon" 主体区域
className "current" 切换时追加的样式
tabEvent "click" 触发事件 可以换成mousemove
index 0 默认第一个为打开 默认当前状态索引（从0开始）
